How to install this

```bash
git clone https://github.com/SleepTheGod/EmailBomb/
cd EmailBomb
pip install -r requirements.txt
python app.py
```
Now Go to the following
http://127.0.0.1:5000/

Usage

```
Mass Email Sender
Recipients (comma-separated emails)
example1@gmail.com,example2@gmail.com
```
